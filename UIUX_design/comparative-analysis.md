# Comparative Analysis

## Project context
My project is an RDI platform that connects companies and students for research and development projects.

## Method choice
My project is at an early stage and no system currently exists. Comparative analysis is suitable because it helps identify common patterns, expectations, and usability issues in existing RDI and talent platforms, providing a strong foundation for designing my own system.

## Systems compared
- Platform A (student project platform)
- Platform B (talent / recruitment platform)
- Platform C (freelancer or collaboration platform)

## Key findings
- Common ways of presenting student skills
- How companies describe project requirements
- Usability issues or missing features

## How this informs my design
This analysis helps define the core features and information structure of my RDI platform.
